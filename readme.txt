Код ПП: 3.140.968
Код ПП в сборе: 2.141.183

В КЭБ плата идет как повышенной сложности (зазоры на DD1 0.15мм). 
Так же на класс точности влияют переходные отверстия, в КЭБ в данном случае не стали увеличивать диаметр отверстия для VIA.

Проблемы:
1. Для VD20 маленькое посадочное место. VD2 и VD6 тоже можно обновить.
2. На сборочнике надо убрать треугольники полярности у диодов VD16,VD18,VD20.

Изменения:
1. Исправленоо подклчение разъема X2 JTAG к DD1 (были перепутаны сигнальные линии).
2. Названия цепей на DD1.11 и DD1.13 поменяны друг с другом, чтобы соответствовали надписям на корпусе.
3. Сигналы на переключатель SW1 переименованы:
	SW_MCLK ->SW_x26dB
	SW_LP2 -> SW_LP1 
	SW_LP1 -> SW_LP0
	SW_LP0 -> SW_RESERVE
4. Добавлены разъемы на землю JP1, JP2.
5. Для полигонов питания +5B, +5B_FI, +3.3B_CPLD, +3.3B добавлены дополнительные конденсаторы.
6. Добавлены нормальные элементы логотипа компании и места для гравировки.
